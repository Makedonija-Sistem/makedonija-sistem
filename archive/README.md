## GitHub Pages serving

Фајлот е преименуван со екстензија `.html.disabled` за да:

- **НЕ се serve** како HTML страница од GitHub Pages
- Корисниците **не можат** случајно да го отворат во прелистувач
- Се чува на disk само како **internal reference**

GitHub Pages serve-ува само:
- `.html` фајлови
- `.htm` фајлови
- Други познати MIME типови

Фајл со екстензија `.disabled` ќе врати **404 Not Found** ако се обидеш да го отвориш директно.
