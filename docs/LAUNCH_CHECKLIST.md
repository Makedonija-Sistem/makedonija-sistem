## CRITICAL · Мора пред публичен launch

### Технички

- [ ] **Сите PWA референции работат** — не 404 грешки
  - Manifest.json валиден ✅ (направено)
  - Service Worker pre-cache само постоечки ✅ (направено)
  - apple-touch-icon точна ✅ (направено)
- [ ] **Firebase placeholder неактивен** — корисникот не е во заблуда
  - FIREBASE_ENABLED = false ✅ (направено)
- [ ] **README точен** — не спомнува непостоечки фајлови ✅ (направено)
- [ ] **404.html постои** ✅
- [ ] **GitHub Pages workflow работи**

### Правни

- [ ] **Placeholder-и заменети со реални податоци:**
  - [ ] [ИМЕ НА ПРАВЕН СУБЈЕКТ]
  - [ ] [АДРЕСА]
  - [ ] [EMAIL ЗА КОНТАКТ]
  - [ ] [ОДГОВОРНО ЛИЦЕ]
- [ ] **Privacy policy** прегледана од правник
- [ ] **Terms of Service** прегледани од правник
- [ ] **Статут** прегледан од правник
- [ ] **Кодекс на интегритет** прегледан од правник
- [ ] Сите legal/ документи **финализирани и без DRAFT етикета**

### Безбедност

- [ ] **Нема чувствителни податоци** во URL parameters
- [ ] **HTTPS задолжителен** (GitHub Pages автоматски)
- [ ] **localStorage** не складира чувствителни податоци
- [ ] **mailto: линкови** не разоткриваат податоци

### Содржински

- [ ] **Бројки означени како цели**, не реалности (на пр. „1,000 основачи" → „цел: 1,000 основачи")
- [ ] **Сите forms имаат disclaimer** „Оваа функција е во подготовка"
- [ ] **Нема измислени имиња на основачи**
- [ ] **Нема измислени партнери** (освен формално согласени)
- [ ] **Нема тврдења за регистрација** ако не е извршена
